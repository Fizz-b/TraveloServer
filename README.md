# Travelo Server




## Installation

Install mongodb compass 
```console
https://www.mongodb.com/try/download/shell
```


Install library:

```console
$ npm install
```

Init database:

```console
$ npm run seed
```

## Document API
https://documenter.getpostman.com/view/20595460/2s8YeuJVTS

## Start Server

```console
$ npm start
```

Start the server (dev):

```console
$ npm run dev
```
